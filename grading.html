BIT UOM Grading System (New Syllabus 2024)

Grading System Overview

The grading system at BIT University of Moratuwa (UOM) uses letter grades based on the Grade Point System. Below is a table that shows the corresponding benchmark percentage and grade points for each grade.

Table 1: Grading System
Benchmark Percentage Grade Grade Point Description
85 and above A+ 4.00 Excellent
75 to 84 A 4.00 Excellent
70 to 74 A- 3.70 Good
65 to 69 B+ 3.30 Good
60 to 64 B 3.00 Good
55 to 59 B- 2.70 Fair
50 to 54 C+ 2.30 Pass
45 to 49 C 2.00 Pass
40 to 44 C- 1.70 Weak Pass
35 to 39 D 1.00 Conditional Pass
34 and below I 0.00 Incomplete
- F 0.00 Fail
- T - Absent
- X - Exemption granted

Important Notes:

  • Grade D or above is required to earn credit for a course module.
  • A student receiving an "F" grade must repeat both continuous assessment (CA) and the end-of-semester examination (SE) components.
  • Grade "T" signifies that a student has successfully completed the continuous assessment component but has been absent for the final examination. The student is allowed to sit for the examination as a first-attempt candidate at a subsequent sitting.
  • By repeating only the end-of-semester examination (SE), the grades I, D, or C- can be improved up to a C grade and considered for calculating the Semester Grade Point Average.
